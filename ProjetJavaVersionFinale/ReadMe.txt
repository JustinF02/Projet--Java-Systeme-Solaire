-----------------------------
Pour éxécuter le programme :

Ouvrir un terminal dans le répertoire courant du fichier SystemeSolaire.jar.
S'assurer que le dossier ressources soit présent dans ce même répertoire.
Dans le terminal, utiliser la commande suivante :
java -jar SystemeSolaire.jar

-----------------------------
Pour créer un fichier .jar :
Sur Intelidji :
->Onglet File -> project structure...
->Artifacts -> appuyer sur le bouton "+"
->Sélectionner l'onglet "JAR" puis "from modules with dependencies"

Une fois sur la fenêtre, il suffit de sélectionner le main systemeSolaire dans le label Main Class.
Appuyer sur OK.

->Onglet build -> Build artifacts
->Nom du .jar -> Build

Le fichier .jar est généré, pour l'executer, renseignez vous dans le helper ci-dessus.
